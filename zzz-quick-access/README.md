# Quick Access - Frequently Used Scripts

This folder contains shortcuts and launchers for the most commonly used scripts.

## Quick Launch Scripts

- **`start-all.ps1`** - Master launcher with menu for all services
- **`start-backend.ps1`** - Start Flask backend (shortcut)
- **`start-ngrok.ps1`** - Start ngrok tunnel (shortcut)
- **`start-flutter.ps1`** - Start Flutter dev server (shortcut)

## Usage

Double-click any script or run from PowerShell:

```powershell
.\zzz-quick-access\start-all.ps1
```

The `start-all.ps1` script provides an interactive menu to start any service.
