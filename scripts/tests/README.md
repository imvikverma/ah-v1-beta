# Test Scripts

This folder contains test scripts for validating broker integrations and API functionality.

## Files

- `test_hdfc_*.py` - Python test scripts for HDFC Sky API
- `test_hdfc_*.ps1` - PowerShell test scripts for HDFC Sky API
- `test_authorise_direct.ps1` - Test HDFC Sky authorise endpoint

## Usage

Run these scripts to test broker API integrations and troubleshoot issues.

```powershell
# Example: Test HDFC Sky credentials
python scripts/tests/test_hdfc_credentials.py
```
